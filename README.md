# Proyecto-de-Desarrollo-Fullstack-Final
Caso Consulta Medica y Dental

Integrantes: Rayén Aguayo M y Javiera Faundez B

Descripción de Proyecto: El proyecto fue pensando para ayudar a una Consulta Medica y Dental, ya que nos dimos cuenta que necesitaban varias cosas para tener un buen servcicio ya que funcionaba bien pero faltaba algunas cosas que mejorar para que no tengan ningun problema y puedan mejorar su atención

Funcionalidades Implementadas: Agregamos 10 microservicios y son medico, paciente, registro de materiales, facturación y presupuesto, ficha medica, reserva y anular hora, receta medica, opinion del paciente, registro de atenciones, pagos
